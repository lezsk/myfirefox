# Firefox的插件


## 浏览相关

Name                | 描述
--------------------|-----------------
FoxyProxy Standard  | 你懂的
Quick Translator    | 翻译工具
Tree Style Tab      | 将tab竖立起来显示
Ever Sync           | 书签同步备份

#### FoxyProxy Standard

> 我不爱用VPN之类的全局代理工具
你只要用 [shadowsocks][1] 或者 [make-proxy][2] 就可以方便的控制哪些翻，
哪些不翻

#### Quick Translator

> 翻译，简单好用。页面中选取要翻译的单词/句子/段落， 
然后点击弹出的小图标，就会在页面的右下角弹出翻译结果。
**注意**： 它使用的 `translate.google.com` 地址，所以最好
开启代理，结合 FoxyProxy 一起使用效果最佳

#### Tree Style Tab

> 浏览器的tab默认都是横向平铺的，当开的tab多了，是很难找到一个特定tab的
这个插件可以将tab竖着显示，并且保留了父tab，子tab的关系。

![tree tab][3]


#### Ever Sync

> 和firefox本地书签结合的很好，
 本地添加新书签后，自动同步。
 当在一个另外的环境中使用firefox的时候，
 只要也安装这个插件，登录帐号后，就可以把书签再同步到本地
